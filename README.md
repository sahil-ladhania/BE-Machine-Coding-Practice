# BE Machine Coding Practice

A personal practice repo for backend machine coding — covering Node.js core modules and REST API development with Express.

## Structure

```
BE Machine Coding/
├── api-practice/               # Express.js REST API practice
│   └── src/
│       └── app.js              # Entry point — Express server on port 3000
└── Core NodeJS Modules Practice/  # Practice with built-in Node.js modules
```

## Projects

### api-practice
Express.js (v5) setup for practicing REST API development.

**Run:**
```bash
cd api-practice
npm install
npm run dev
```
Server starts at `http://localhost:3000`

### Core NodeJS Modules Practice
Sandbox for experimenting with Node.js built-in modules (fs, path, http, events, streams, etc.).

## Tech Stack
- Node.js
- Express.js v5
